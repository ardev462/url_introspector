URL Introspector

Summary

This extension replaces 'raw' web URLs with human readable hyperlinks using text contained within the URL itself and a few simple rules.

This works best on web sites that follow a convention for page naming (as many do) whereby a description of the page content is used to create the page name (or breadcrumb path) together with additional characters such as spaces or underscores to delineate words.

URL Introspector allows you to define what parts of the URL you want to use in your hyperlink text and to control some of the formatting such as capitalisation and token replacement in the link text

Configuration settings are saved to browser storage so your previous choices will remain until you change them.

Options

Text Formatting

Capitalise the first letter of every word in the hyperlink text

Character Substitution

Replace + _ or - characters with spaces in hyperlink text

Text Template

Given a URL of the format : https://www.abcde.com/3/2/1/filename.html

Where 3,2,1 are postitional placeholders for the text in the corresponding part of the URL and the special placeholder 'filename' denotes the name of the file without the extension...
